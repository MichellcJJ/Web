1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para dar estilo y diseño a las páginas web. Controla colores, tamaños, márgenes, posiciones, fuentes, fondos y toda la apariencia visual del contenido HTML.

2. ¿Cómo utilizar CSS con HTML?

CSS se integra con HTML para aplicar estilos a los elementos de la página. Puede usarse de tres maneras: en línea, interno o externo. Cada método permite definir cómo se mostrará el contenido, ya sea directamente en la etiqueta, dentro del documento o desde un archivo aparte.

3. CSS en línea

El CSS en línea se aplica directamente dentro de una etiqueta HTML usando el atributo style. Sirve para dar estilo rápido a un elemento específico sin afectar a los demás.

4. CSS interno.

El CSS interno se escribe dentro de la etiqueta <style> ubicada en la sección <head> del documento HTML. Este método permite aplicar estilos a toda la página sin utilizar un archivo externo.

5. CSS Externo

El CSS externo se coloca en un archivo aparte con extensión .css y se enlaza al documento HTML mediante: <link rel="stylesheet" href="estilos.css"> Es la forma más organizada y profesional, porque separa el contenido del diseño.
