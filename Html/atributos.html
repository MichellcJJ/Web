1. -Atributos globales.

Primer subtema que forma parte de los atributos globales; suele representar una propiedad o punto que amplía esa categoría.

1.2:

1.3:
Tercer subtema dentro de los atributos globales, que detalla otra parte del mismo grupo.

3. -Atributos input.

Son todas las características que permiten controlar cómo funciona un campo de entrada dentro de un formulario. Estos contenidos explican cómo un input puede aceptar diferentes tipos de datos, cómo puede validar lo que el usuario escribe, cómo puede mostrar mensajes, limitar valores, activar opciones automáticas y definir comportamientos específicos.
