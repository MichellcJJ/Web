1.- Etiquetas de documento

1.1 html:
Etiqueta principal que encierra todo el contenido de una página web. Le dice al navegador que el documento está escrito en HTML.

1.2 head:
Contiene información que no se muestra directamente, como estilos, scripts, metadatos y el título de la pestaña.

1.3 body:
Todo lo que el usuario ve en la pantalla: texto, imágenes, tablas, botones, etc.

2.- Etiquetas de Meta Datos

2.1 meta:
Proporciona información sobre la página: codificación (UTF-8), descripción, autor, palabras clave, etc. Ayuda al SEO y al navegador.

2.2 style:
Permite escribir código CSS directamente dentro del documento HTML para aplicar estilos.

2.3 title:
Texto que aparece en la pestaña del navegador. También lo usan los buscadores como título del resultado.

2.4 base:
Establece una URL base, para que los enlaces e imágenes del documento usen esa ruta por defecto.

2.5 link:
Conecta archivos externos con el documento, por ejemplo, hojas de estilo CSS o iconos.

3.- Etiquetas de Agrupación

3.1 p:
Representa un párrafo de texto. Se usa para separar bloques de contenido.

3.2 hr:
Dibuja una línea horizontal que sirve para dividir secciones visualmente.

3.3 ul:
Crea una lista desordenada, con viñetas o puntos.

3.4 div:
Es un contenedor general que se usa para agrupar elementos y aplicarles estilos o posicionamiento.

3.5 figure:
Contenedor para imágenes, gráficos, códigos o elementos visuales que van acompañados de una descripción.

3.6 figcaption:
Texto que describe la imagen o contenido dentro de una figure. Funciona como un pie de foto.

4.- Etiquetas Textuales

4.1 a:
Crea enlaces. Puede dirigir a otra página, un archivo, una sección interna o incluso iniciar descargas.

4.2 span:
Contenedor en línea (no ocupa toda la fila). Se usa para aplicar estilos o resaltar partes dentro de un texto.

4.3 sub:
Muestra texto en subíndice (más abajo). Se usa en fórmulas químicas o notas.

4.4 sup:
Muestra texto en superíndice (más arriba). Se usa para exponentes, referencias o notas.

4.5 cite:
Se usa para títulos de obras, libros, artículos, películas o citas formales.

5.- Etiquetas Semánticas

5.1 nav:
Agrupa enlaces de navegación, como menús o barras superiores.

5.2 section:
Crea secciones temáticas dentro de la página. Útil para dividir contenido en partes organizadas.

5.3 article:
Contenido independiente y completo, como un post, una noticia o una tarjeta de información.

5.4 header:
Encabezado de la página o de una sección. Suele contener el título, logo o menú.

5.5 main:
Área principal del sitio, donde va el contenido más importante. Solo debe haber uno por página.

5.6 footer:
Pie de página: información final como derechos de autor, contactos, términos o enlaces secundarios.
