¿Qué es HTML?

Es el lenguaje estándar para crear y estructurar páginas web. Define el contenido y la estructura del contenido de una página, como texto, imágenes, enlaces, tablas y más, mediante un sistema de etiquetas que los navegadores web interpretan para mostrar la información.

Historia HTML.

fue inventado por Tim Berners-Lee en 1980 para compartir documentos y crear la World Wide Web. La primera versión oficial, HTML 1.0, se publicó en 1991. Desde entonces, el lenguaje ha evolucionado a través de versiones como HTML 2.0 (1995) y HTML 4.0 (1997), que introdujo la separación de contenido y presentación (CSS), hasta la versión actual, HTML5 (lanzada en 2014), que incorpora soporte multimedia, etiquetas semánticas avanzadas y mejoras para aplicaciones web complejas.

Editores de HTML.

son software para crear y editar código HTML. Hay dos tipos principales: editores de texto, como Visual Studio Code y Sublime Text, que permiten codificar directamente; y editores WYSIWYG (What You See Is What You Get), como Adobe Dreamweaver, que ofrecen una interfaz gráfica visual. Otros editores populares incluyen Notepad++, Atom, Brackets y Bluefish.

Tu primer documento HTML.

<!DOCTYPE html> <html> <head> <title>Mi Primera Página de Gatos</title> </head> <body> <h1>Página de Gatos </h1> <p>Este es mi primer documento HTML sobre gatos. Usé etiquetas básicas para crear mi contenido.</p> <p>Los gatos son animales increíbles & especiales. Aquí muestro una imagen:</p> <img src="gato.jpg" alt="Gato tierno" width="250"> <p>Si quieres ver más, haz clic aquí: <a href="mas-gatos.html">Ver más gatos »</a> </p> </body> </html>

¿Que son Etiquetas en HTML?

son códigos que se usan para estructurar el contenido de una página web, indicándole al navegador cómo presentar la información. Funcionan como marcadores que encierran el contenido, definiendo su propósito, como títulos (\(<h1-h6>\)), párrafos (\(<p>\)), enlaces (\(<a>\)) o imágenes (\(<img>\))

¿Que son Atributos en HTML?

son palabras especiales que se añaden a las etiquetas de inicio para proporcionar información adicional o modificar el comportamiento de un elemento HTML. Se componen de un nombre y un valor, y se colocan dentro de la etiqueta de apertura.

Reglas fundamentales en HTML.

Las reglas fundamentales de HTML incluyen la estructura básica del documento (<!DOCTYPE html>, <html>, <head>, <body>), el uso correcto de etiquetas (abrir y cerrar correctamente, usar minúsculas) y la organización del código a través de sangrías y anidación. También es importante seguir las convenciones de formato, como el uso de comillas dobles para los atributos y la validación del código para asegurar su compatibilidad.

Etiquetas Para hacer un documento HTML.

Las etiquetas principales para crear un documento HTML son <!DOCTYPE html>, <html>, <head> y <body>. El <!DOCTYPE html> declara el tipo de documento, <html> es el elemento raíz, <head> contiene metadatos como el título (<title>), y <body> contiene el contenido visible de la página.
